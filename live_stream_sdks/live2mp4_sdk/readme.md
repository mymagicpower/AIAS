## SDK功能
rtsp/rtmp 拉流录制视频
![result](https://djl-model.oss-cn-hongkong.aliyuncs.com/AIAS/video_sdk/mp4.png)

## 运行例子
1. 首先下载例子代码
```bash
git clone https://github.com/mymagicpower/AIAS.git
```

2. 导入examples项目到IDE中：
```
cd live_stream_sdks/live2mp4_sdk
```

3. 运行例子代码：LiveStream2MP4


## 效果如下：
录制视频保存于
```
build/output/live.mp4
```
